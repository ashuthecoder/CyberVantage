Welcome to CyberVantage

Cybersecurity Training and Simulation Platform (API)

Available API Endpoints

Authentication

GET/POST /api/auth

  • POST /api/auth/register
  • POST /api/auth/login
  • GET /api/auth/me

Learning

GET /api/learning

  • GET /api/learning/modules
  • GET /api/learning/content/<module_id>
  • POST /api/learning/progress

Simulation

GET/POST /api/simulation

  • GET /api/simulation/scenarios
  • POST /api/simulation/start/<scenario_id>
  • POST /api/simulation/submit/<simulation_id>

Analysis

GET/POST /api/analysis

  • GET /api/analysis/performance
  • GET /api/analysis/recommendations
  • POST /api/analysis/certificate

Threats

GET/POST /api/threats

  • POST /api/threats/url
  • POST /api/threats/file
  • GET /api/threats/intel

Note: This v2 app exposes API endpoints. For the full web UI, run the main application.
