Multi-Hop Link Analysis

What are multi-hop links?

Multi-hop links are redirect chains that take users through multiple URLs before landing on the final destination. Attackers use these redirect chains to:

  • Evade detection by security tools
  • Hide the actual malicious URL behind legitimate-looking domains
  • Create complex obfuscation techniques to avoid scanners
  • Track user clicks through multiple analytics platforms
  • Dynamically serve different content based on user attributes

Our Deep Search technology follows every redirect in the chain, scanning each hop for potential threats, and provides comprehensive visibility into the entire redirect path.

Example Chain
  1. User clicks link in email
  2. Redirects to tracking domain
  3. Redirects to legitimately hosted page
  4. JavaScript redirects to phishing site
  5. Final malicious destination

Basic scanners only check the first URL!
